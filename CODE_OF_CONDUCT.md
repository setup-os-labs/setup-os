# Code of Conduct

Setup OS follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of positive behavior:

- being respectful and direct
- giving actionable feedback
- assuming good intent
- focusing critique on ideas and code
- helping contributors understand project safety boundaries

Unacceptable behavior includes harassment, threats, personal attacks, and publishing private information without permission.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or block participants who violate these standards.
