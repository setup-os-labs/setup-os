# Packages

Shared Setup OS packages live here while the system remains in one monorepo.

- `core/`: orchestration primitives and common domain models.
- `schemas/`: versioned public schemas for specs, manifests, and evolution proposals.
- `agent-runtime/`: typed agent execution helpers.
- `conversation-import/`: source adapters and conversation normalization.
- `blueprint-engine/`: generated-system template rendering.
- `component-registry/`: OSS discovery, scoring, and registry metadata.
- `policy/`: risk gates, permissions, and policy evaluation helpers.
