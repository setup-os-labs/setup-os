# Learning OS Conversation Guide

Use this when planning a local learning or skill-building assistant.

## Goal

What skill, credential, project, or body of knowledge should the system help with?

## Inputs

What materials can it read: books, notes, courses, bookmarks, transcripts, PDFs, or project repos?

## Learning Style

What cadence, difficulty, practice style, and feedback loop work best?

## Workflows

Should it create plans, quizzes, flashcards, project tasks, reviews, or progress reports?

## Alerts

What should it remind the user to do?

## Existing Tools To Check

Ask whether LMS tools, spaced repetition apps, note systems, or coding environments can be composed.
