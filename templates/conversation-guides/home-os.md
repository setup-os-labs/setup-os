# Home OS Conversation Guide

Use this when planning a local home operations assistant.

## Goal

What home workflows should the system make easier?

## Inputs

List devices, manuals, warranties, bills, maintenance schedules, vendors, and household preferences.

## Workflows

Should it track chores, maintenance, utilities, supplies, repairs, projects, or budgets?

## Alerts

What reminders or risk alerts should it send?

## Action Model

Should it only advise, draft vendor messages, create calendar events, or automate smart-home actions?

## Existing Tools To Check

Ask whether Home Assistant, calendars, task tools, or document stores can be composed.
